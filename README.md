# Krita Comic Guides

A small Krita Python plugin that:

1. Creates a new document:  **11" × 17", @ 600 PPI** 
2. Adds bleed, trim and safe guides.
3. Creates a 9-panel vector grid.

## Install


## Usage

From the menubar: **Tools → Scripts → New US Comic Page (11×17)**.
